- **CUDA Image Processing** *(May 2026 – present)* — Dual-path CUDA C++ image-processing pipeline for automated coin classification. Implements morphological erosion and dilation with selectable 3×3 and 11×11 kernels, Sobel edge detection, and Connected Component Labeling via GPU-accelerated union-find. ### ⚙️ GPU / CUDA

- **CUDA Research Project** *(Apr 2026)* — Implemented Newton's divided-difference quadratic interpolation in CUDA C++, benchmarking three execution strategies: 100% host, 100% device, and a hybrid approach computing coefficients on the host before dispatching the evaluation kernel using shared memory and tiling. Parallelized curve evaluation across up to 50M points using a 1D grid of 256-thread blocks; the hybrid experiment eliminated shared-memory synchronization overhead by passing scalar coefficients directly as kernel arguments.
